WEB DE ZIA ROSE

Esta carpeta contiene una mini web tipo Linktree lista para publicar en Cloudflare Pages.

ARCHIVOS IMPORTANTES

- index.html: textos y enlaces de la pagina.
- style.css: colores, botones y diseno.
- script.js: boton para copiar el enlace de la pagina.

COMO SUBIRLA A CLOUDFLARE PAGES

1. Sube estos archivos a un repositorio de GitHub.
2. En Cloudflare ve a Workers & Pages.
3. Crea un proyecto nuevo de Pages.
4. Conecta el repositorio.
5. Usa esta configuracion:

Framework preset: None
Build command: dejar vacio
Build output directory: .

6. Pulsa Deploy.

Cloudflare te dara una URL parecida a:

https://web-de-zia-rose.pages.dev
